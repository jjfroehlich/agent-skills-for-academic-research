# Manuscript Feedback Report

Use this as an output pattern for integrated manuscript feedback.

## Pattern

```text
Scope: manuscript draft for <audience/destination>.
Selected lenses: scientific-writing; add data-visualization-and-figures or publishing-and-peer-review if relevant.

What works:
- <strength to preserve>

Must fix first:
1. <issue -> action -> rationale>

Major revisions:
1. <section/claim/evidence issue -> specific revision>
2. <figure/table/legend issue -> specific revision>

Secondary improvements:
- <style, flow, or consistency issue>

Missing context:
- <journal, figure, data, or audience detail that would change advice>

Next revision step:
- <ordered action>
```

## Weak vs strong

Weak: "The paper is hard to follow and needs better flow."

Strong: "The Results section reports methods chronology before findings. Reorder each subsection around finding -> evidence -> local conclusion, then move interpretation-heavy sentences to the Discussion."

## Bad assistant behavior

- Rewriting paragraphs before diagnosing section function.
- Treating figure problems as prose problems only.
- Listing every minor wording issue before the central claim is stable.
