# Grant Writing Templates

Use these templates when the user asks for a structured review, rubric, table, or rewrite plan.

## Funder-Fit Preflight

```text
Funder and mechanism:
Call scope:
Applicant eligibility:
Required sections:
Template and formatting constraints:
Review criteria:
Budget cap and allowed costs:
Internal deadline:
Evidence still needed:
Decision: proceed / retarget / clarify
```

## Active Premise And Why Questions

```text
Objective:
Active premise:
Why this problem:
Why now:
Why this approach/model/system:
Why this applicant/team/environment:
Expected change if successful:
```

## Approach Block

```text
Aim or work package:
Rationale:
Overall plan:
Specific example or preliminary data:
Interpretable outcomes:
Potential issue:
Backup technique or workaround:
```

## Budget Feasibility Check

```text
Promised work:
Required personnel/resources:
Requested funds:
Allowed and disallowed costs:
Delivery risk if underfunded:
Rescope decision:
```

## Risk-Return-Feasibility Row

```text
Ambitious claim:
Expected scientific return:
Scientific risk:
Execution risk already controlled:
Feasibility evidence:
Reviewer doubt to address:
```

## Bold-Statement Skim Test

```text
Read only headings:
Read only topic sentences:
Read only figure titles/captions:
Read only bolded statements:
Can the reviewer recover the proposal gist?
Statements to rewrite or unbold:
```
